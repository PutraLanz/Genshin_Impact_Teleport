# 如何使用:

>**看文件夹名字中[]内的内容：**
>
>**ver:数据版本**
>
>**AL:范围**
>
>**YO:Y轴偏移**
>
>**2023-01-01:日期**
>
>**CN/EN:语言**
>
>>**例如：①Teleport ALL[v3.4][AL-15M][YO-1M][2023-01-18][CN]**
>>
>>**Teleport All /3.4版本/基于15M范围效率优化/2023年1月18日更新/中文**
